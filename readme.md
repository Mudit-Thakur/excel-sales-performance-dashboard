# 📊 Sales Performance Dashboard (Excel)

## 🔍 Overview

This project focuses on building a dynamic and interactive **Sales Performance Dashboard** using Microsoft Excel.

The goal is to transform raw transactional data into meaningful business insights that help stakeholders understand performance trends, identify revenue drivers, and detect potential issues.

---

## 🎯 Business Problem

Most dashboards only answer:

> “What happened?”

This project goes further by answering:

* Why did revenue change over time?
* Which categories are driving growth or decline?
* Which regions are underperforming?

---

## 🧩 Key Features

* Interactive **Slicers** (Region, Category, Month)
* KPI Metrics:

  * Total Revenue
  * Total Orders
  * Average Order Value (AOV)
* Trend Analysis (Monthly Revenue)
* Category-wise Performance Breakdown
* Clean, print-ready layout for reporting

---

## 🛠️ Tools & Skills Used

* Microsoft Excel

  * Pivot Tables
  * Slicers
  * Data Cleaning
  * Dashboard Design
* Basic Data Analysis
* Business Insight Generation

---

## 📈 Key Insights

Example insights derived from the dashboard:

* Revenue dropped by **~18% in March**, primarily driven by decline in Electronics category
* West region underperformed compared to other regions
* High discounts correlated with lower average order value

---

## 📁 Dataset

The dataset includes:

* Order-level sales data
* Product categories
* Region information
* Pricing and discount details

(Data was cleaned and standardized before analysis)
